GitHub Desktop method — easiest way, no terminal, no commands needed

Phase A — Set up GitHub (one time only)
1
Create a free GitHub account
Skip if you already have one
1
Go to github.com → click "Sign up"
2
Enter email, password, username → verify your email
Open GitHub Signup →
2
Install GitHub Desktop
A simple app — drag & drop your files, no commands needed
1
Click the button below to download GitHub Desktop
2
Install it and sign in with your GitHub account
Download GitHub Desktop →
3
Create a repo and upload your project
This puts all your files on GitHub
1
Open GitHub Desktop → click "Create a New Repository on your hard drive"
2
Set the name to manasa-events — remember the Local Path (the folder it creates)
3
Click "Create Repository"
4
Open File Explorer / Finder → go to the Local Path folder that was just created
5
Copy all your MANASA project files into that folder (select all → Ctrl+C → paste into the new folder)
6
Go back to GitHub Desktop — you'll see all files listed on the left side
7
Type first upload in the Summary box at bottom left → click "Commit to main"
8
Click "Publish repository" at the top → make sure "Keep this code private" is unchecked → click Publish Repository
✅ Your project is now on GitHub! You can check it at github.com/YOUR-USERNAME/manasa-events

⚠️ Free plan note: Render's free plan sleeps after 15 minutes of no visitors. The first load after sleeping takes about 30 seconds — this is normal. After waking up, everything works fast.
